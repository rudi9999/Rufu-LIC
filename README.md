# Rufu-LIC

## instalacion

wget --no-cache -O /usr/bin/install-LIC https://github.com/rudi9999/Rufu-LIC/raw/main/install-LIC; chmod +x /usr/bin/install-LIC; install-LIC

## instalar una nueva licencia

solo escrive el comando: intall-LIC -r
